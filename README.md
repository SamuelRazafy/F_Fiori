# Mon application Fiori - zlistperssam / F_Fiori

Application SAPUI5/Fiori pour la liste des personnes SAM.

Développée avec SAP Fiori tools dans VS Code.

Voici le code source de notre exercice. Vous pouvez le consulter et l’utiliser comme référence pour appliquer les mêmes principes dans votre propre exercice.

Concernant le CRUD, nous avons implémenté seulement les opérations Read, Create et Delete. La partie Update n’est pas encore finalisée, donc c’est à vous de la compléter afin de terminer l’exercice.

Dans cet exemple, vous pourrez également bien comprendre l’utilisation d’un fragment, qui est un élément important dans le développement d’applications Fiori.

Nous considérons que vous êtes tous des ingénieurs, donc ce ne sera certainement pas très difficile pour vous de comprendre la logique et la structure du code.

Lors de notre prochaine session, nous vous expliquerons plus en détail le fonctionnement de ce code.

Bon courage à tous et essayez de terminer l’exercice.

Nous restons bien sûr disponibles si vous avez des questions.
